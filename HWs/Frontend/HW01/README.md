# 前端作业(Day1)
欢迎来到前端作业Day1, 今天你有如下任务(5-30min 完成), 你需要在明天之前完成作业.

## 1.0


1. 编写一个基础的HTML文件
2. 标题为: `Hello World`
3. 内容为一段文字, 居中显示(课上没有说如何居中显示), 段落内容为`Hello World!`
4. 提示: 课上已经教如何编写标题和段落, 居中段落时需要用到段落(p标签)的一个属性align, 你可以自行在网上搜索
